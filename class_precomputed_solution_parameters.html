Detailed Description

The PrecomputedSolutionParameters class stores parameters for the 'Precomputed_solution' XML element used to read in the data from a precomputed solution for the simulation state.

<Precomputed_solution>
<Project_from_face> lumen_wall </Project_from_face>
</Precomputed_solution>
